INTRODUCTION

Crime is an inevitable part of any developing country and cyber-crime is a branch of crime that has been uprooted along with the rapid growth of the internet. Cybercrime is any criminal activity that involves a computer, networked device or a network. The cyber-crimes include tampering computer source documents, computer related offences, ransom-ware, identity theft, violation of privacy, cyber terrorism, publication of obscene/sexually explicit act in electronic form, publishing material depicting children in sexually explicit act in electronic form, abetment of suicide(online), cyber stalking/bullying of women/children, online banking fraud. This database will consist of cyber-crimes committed state/UT-wise persons arrested for cyber-crime in the year 2020 under all crimes mentioned above.




Functionalities of the system

As mentioned in introduction the number of cyber-crimes committed has been rapidly increasing because of the increase in availability of computer and network to common man. To prevent the increase of cyber-crimes happening in society we need a deeper understanding about the crimes committed, how they are committed, why they are committed and how it changes over time. All this can be achieved by analyzing a good data set of cyber-crime. The main goal of this database is to store data about cyber-crimes in an efficient way so it can be analyzed and provide useful insights to prevent cyber-crimes.
